## TL;DR

- 跨境电商短视频痛点：素材生命周期短（2-3 周）、制作成本高（1000 元/分钟）、本土化要求高
- 多 Agent 协同：编导 Agent + 剪辑 Agent + QA Agent，7 天工作压缩到 3-4 小时
- 核心方法：原子能力解耦 + 垂直行业知识库（100 万条素材，13 个品类）
- 效果：转化率提升 40%，成本降低 90%

---

## 行业痛点

### TikTok/YouTube Shorts 短视频营销挑战

1. **素材生命周期短**：优质素材 2-3 周失效，大卖每天需投放 200+ 条视频
2. **制作成本高**：传统团队 5-7 天，约 1000 元/分钟
3. **本土化要求高**：语言、文化、审美差异大，需覆盖多语种
4. **原创性要求**：平台去重算法严格，简单混剪难获流量

### 短视频成功要素

- **黄金三秒法则**：前 3 秒决定 90% 留存
- **爆款框架**：钩子 + 卖点展示 + CTA 引导
- **持续迭代**：100 条素材可能只有 1-2 条爆款

---

## 多 Agent 架构

### 三大核心 Agent

| Agent | 职责 |
|-------|------|
| 编导 Agent | 生成脚本、故事板，描述每个镜头的时长/运镜/文案 |
| 剪辑 Agent | 精剪工作，按爆款框架匹配素材 |
| QA Agent | 检测脚本是否符合爆款框架（训练 100 万条素材） |

### 五层能力架构

```
生产执行层 → 分镜生成、AI 匹配、合成流水线、质量控制
内容生产层 → 跨模态脚本生成、多语言本地化引擎
智能分析层 → 视频素材分析、爆款视频分析
数据准备层 → 达人素材、爆款素材、电商素材、AI 生成素材
模型接入层 → OpenAI、Gemini、Deepseek、豆包
```

### 核心处理流程

1. **视频分析 Agent**：拆解参考视频，识别脚本结构
2. **脚本生成 Agent**：基于商品信息生成新脚本
3. **素材匹配 Agent**：Embedding 相似度匹配素材库
4. **视频合成 Agent**：合成 15-20 秒成片，支持在线微调

## 关键技术实现

### 数据采集

- 浏览器插件：刷 TikTok 时一键采集爆款视频
- 链接抓取：输入亚马逊链接自动抓取图片、描述
- 用户上传：商品信息和卖点描述

### 智能分析

- 视频自动切片、打标、入库
- 对每个片段进行图片/视频理解，生成画面描述
- 建立素材库索引

### 素材匹配

- 将故事板描述与素材库描述进行 Embedding
- 在相似度空间中找到最匹配的素材

## 精细化运营

### TikTok 素材成功公式

```
前 3 秒钩子（决定留存）+ 中间片段（拉停留）+ CTA 结尾（促转化）
```

### 通用分镜模板

- 开箱视频：通用钩子，适用于各品类
- 反转内容：如圣诞节摔倒后接促销信息，拉停留效果好

### 垂直知识库

- 25 个行业知识库（3C、美妆、日用、家电等）
- 13 个品类爆款框架
- 本地素材库 + 爆款脚本 RAG
